# ユーザーストーリー（開発優先度つき）

## P0（MVP必須）

- 主催者は10分でイベントを作り、**QRを印刷して入口に置ける**
- 参加者はQRから**即カメラ起動→撮影→フレーム合成→応募確定**できる
- 応募時に**二次利用同意**を取得できる（同意ログが残る）
- 主催者は最低5指標（scan/capture/save/outbound/submit）を見られる
- 応募一覧で「同意あり」だけ絞り込める

## P1（事業化で効く）

- 入口別QR（Aゲート/Bゲート）で効果差を見られる
- ワンタイムQR（チケット/リストバンド）で**1人1回**を強化できる
- 応募作品をワンクリックでDL（同意ありのみ）できる
- キャンペーン規約のテンプレ管理（イベントごとにバージョン保持）

## P2（エンプラ・買収で効く）

- 監査ログ（誰がいつDLしたか）
- 自動レポート生成（PDF/Notion貼り付け）
- 違反/不正検知（同一画像/連投パターン）
