# 画面ごとの必要コンポーネント（実装の抜け漏れ防止）

## 参加者側（/e/{event_token}）

| 画面 | 必要コンポーネント・機能 |
|------|---------------------------|
| U0 LP | 同意チェック（必須）、同意して開始ボタン、既提出時は U5 へ |
| U1 カメラ | getUserMedia、撮影ボタン、撮り直し |
| U2 プレビュー | 合成（Canvas：frame overlay）、「この1枚で応募」「撮り直し」 |
| U3 応募確定 | 応募ボタン（提出1回）、同意チェック、任意で連絡先 |
| U4 保存/シェア | コピー（キャプション）、IG/X 遷移ボタン、計測（save_click / outbound_click） |
| U5 完了 | 提出済み表示、再アクセス時はここへ |

## 管理側（/admin）

| 画面 | 必要コンポーネント・機能 |
|------|---------------------------|
| A0 ログイン | マジックリンク or パスワードフォーム |
| A1 一覧 | イベント一覧、新規作成・ダッシュボード・応募一覧へのリンク |
| A2 イベント作成 | テンプレ選択、10分で完了するフォーム |
| A3 イベント編集 | 文言・ハッシュタグ・同意文言・SNS導線 |
| A4 フレーム管理 | アップロード（署名URL）、プレビュー、差替（active切替） |
| A5 QR管理 | 入口別QR作成、QRコード生成（PNG/PDF） |
| A6 ダッシュボード | 集計API表示（scan/capture/save/outbound/submit） |
| A7 応募一覧 | フィルタ（同意あり）、タグ・メモ、DL（署名URL） |
| A8 エクスポート | CSV ダウンロード |
